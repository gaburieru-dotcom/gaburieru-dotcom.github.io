Tower Defense Epic

サイバー空間を舞台にした王道タワーディフェンス！迫りくる無数の敵ロボットに対し、多様な属性とスキルツリーを備えた防衛タワーをグリッドマップに戦略的に配置。敵の進行経路をプレイヤーが自らデザインする「迷路構築」が勝利の鍵を握る本格ゲームです。

主な特徴：

  • 自由な迷路設計: 敵の移動ルートを引き延ばし、火力を集中させる最適な迷路を構築。
  • 深いシナジー効果: 減速、火炎、範囲雷、高威力レーザーなどタワー同士のシナジーを発揮。
  • リアルタイム進行: 刻一刻と変化する戦況に合わせ、タワーのアップグレードや売却を即座に決断。
  • 対応プラットフォーム iOS / Android
  • 開発エンジン / 言語 Unity / C#
  • 価格 無料 (一部アプリ内課金あり)
  • 最新バージョン v1.2.4 (2026.04)

Utility Fitness

トレリンク (TreLink)

「リンクで共有できる筋トレ記録アプリ」をコンセプトにした、トレーニーのためのログ管理アプリ。作成したトレーニングメニューや履歴をURLリンク経由で友達と簡単に共有・インポート可能。重量やレップ数の入力手順を極限までシンプルにし、ジムでのインターバル中に片手でサッと入力可能です。

主な特徴：

  • URLリンク共有: 自作メニューやトレーニング履歴を簡単なリンクで友達にシェア・インポート可能。
  • 極小タップUI: テンキー入力と履歴コピー機能により、数秒でセット完了を素早く記録。
  • 多機能分析＆グラフ: 部位別の総負荷量グラフや1RM推移などを視覚的にフィードバック。
  • HIITタイマー内蔵: カスタムBGM設定や音声ガイダンス付きの高機能HIIT/タバタタイマーを搭載。
  • 対応プラットフォーム iOS (Android版 開発中)
  • 開発言語 / 技術 Flutter (Dart)
  • 価格 無料 (一部アプリ内課金・広告あり)
  • 最新バージョン v1.0.1

Utility Sports Swift / iOS

タイム測定＆垂直跳び測定

超精密タイム計測

240fpsのスロー動画なら、0.004秒単位の分析が可能。スプリントのタイムや格闘技の動作速度など、あらゆるパフォーマンスを数値化。

垂直跳び解析

滞空時間を計測するだけで、物理演算に基づいたジャンプ高を自動算出。「空中を飛んでいた時間」という確固たる証拠から、真の跳躍力を導き出します。

動画へのデータ焼き付け

計測結果（タイムやジャンプ高）を動画上に合成してエクスポート。プロの分析映像のような仕上がりで、あなたの成果を記録に残せます。

拡大シーク機能

足元やゴールラインをズームしながら1コマずつ送り・戻しができるため、離地や着地の瞬間を1フレームも逃しません。

  • 対応プラットフォーム iOS 17.6以降 (iPhone / iPad)
  • 開発言語 / 技術 Swift (SwiftUI / AVFoundation)
  • 価格 無料 (一部App内課金あり)
  • 最新バージョン v1.0.0

Utility Productivity

FocusTimer

ポモドーロテクニックを活用し、仕事や勉強への集中力を高めるミニマルなWebタイマー。25分の作業と5分の休憩を1サイクルとし、画面を閉じずに音とアニメーションで切り替えを知らせます。ダークテーマとの親和性が高く、環境音再生機能があなたの「ゾーン入り」を優しくサポート。

主な特徴：

  • インストール不要: Webブラウザがあれば、PCでもスマートフォンでも即座に起動可能。
  • 環境アンビエント音: 「雨の音」「静かなカフェ」「ホワイトノイズ」など集中を助ける背景音をミックス可能。
  • データ分析: その日こなしたポモドーロ数と累計集中時間をローカルストレージに自動蓄積。
  • 対応プラットフォーム Web (iOS / Android / PCブラウザ)
  • 開発言語 / 技術 HTML5 / Vanilla CSS / ES6 Javascript
  • 価格 完全無料
  • 最新バージョン v1.0.2
